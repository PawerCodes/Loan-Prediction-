Loan Approval Prediction Using Machine Learning



Project Description


ML-based Loan Prediction system using SVM with 83% test accuracy.



Technologies Used:


Python
NumPy – Numerical operations
Pandas – Data handling
Seaborn – Data visualization
Scikit-learn – Machine Learning
SVM Algorithm

Dataset


Dataset includes applicant details and loan status (Approved/Not Approved) used for training a classification model.



Model Architecture


The project uses a Support Vector Machine (SVM) classifier with a linear kernel.
The dataset is divided into 90% training data and 10% testing data using train_test_split().

Accuracy


Accuracy on training data :  0.7986111111111112
Accuracy on test data :  0.8333333333333334


Conclusion


This project successfully demonstrates the use of Machine Learning for loan approval prediction. The SVM model achieved good accuracy and can help in automating loan decision processes.
